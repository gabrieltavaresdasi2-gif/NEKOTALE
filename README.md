# Nekotale

Site estático pronto para publicação pública.

## Publicar com GitHub Pages

1. Crie um repositório no GitHub.
2. Faça upload da pasta deste projeto para o repositório.
3. No GitHub, abra Settings > Pages.
4. Em Source, escolha Deploy from a branch.
5. Selecione a branch principal e a pasta raiz `/`.
6. Salve e aguarde alguns minutos.

O site será disponibilizado em um link público como:
`https://seu-usuario.github.io/seu-repositorio/`
